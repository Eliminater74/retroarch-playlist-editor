RetroArch Playlist Builder

by Marc Robledo See on GitHub Donate

This web app can generate a valid .lpl playlist file for RetroArch
Drag and drop items/playlists here
    This was built mainly to create my own RetroArch playlists because its built-in scanner does not detect custom content.

    Quick usage:
    1. Drag & drop ROM files here
      You can even drop an existing .lpl playlist!
    2. Use the Tweak button if needed
      The Remove dupes option is very useful when adding a lot of files into an existing playlist
    3. Export and enjoy!
